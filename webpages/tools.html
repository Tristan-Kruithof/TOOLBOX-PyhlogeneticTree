biopython

Biopython is a tool designed for the language python. It can be used to retrieve information from NCBI. The user can specifie the database, then add a search term. Biopython will then return all the results from the search term in the chosen database. We use biopython to retrieve a specific gene in the mitochondrial DNA of species. When you fill in the name of an organism on this website, biopython will search NCBI for the gene sequence of that organism. All the organisms that you want to include in your phylogenetic tree will have this particular gene sequence and this sequence is used to compare the organisms.
